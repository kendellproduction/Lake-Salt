# Gallery Placeholder Images

This directory contains placeholder images for the Lake Salt Bartending gallery. Replace these with actual event photos.

## Image Guidelines:

- **Size**: Recommend 1200x800px for carousel slides, 400x400px for grid items
- **Format**: JPG or PNG
- **File Names**: 
  - `wedding-1.jpg` - Wedding event photo
  - `corporate-1.jpg` - Corporate event photo
  - `private-1.jpg` - Private gathering photo
  - `holiday-1.jpg` - Holiday party photo
  - `private-2.jpg` - Private event photo
  - `lake-1.jpg` - Lake/outdoor celebration photo

## To Add Your Photos:

1. Replace the placeholder images in this directory with your actual event photos
2. Update the Gallery component to reference the actual image paths if different
3. Ensure images are optimized for web (compressed but high quality)

The website is currently using gradient overlays with placeholder text. Once you add photos, you can update the Gallery component to display actual images.
