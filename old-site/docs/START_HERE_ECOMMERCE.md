## ⚡ Super Quick Start (5 Minutes)

### If you just want to see it work:

1. **Create Shopify Store**
   ```
   → Go to shopify.com
   → Sign up (free trial)
   → Choose store name
   ```

2. **Add Test Product**
   ```
   → Products → Add product
   → Title: "Test Cocktail Kit"
   → Price: $45.00
   → Save
   ```

3. **Get API Token**
   ```
   → Settings → Apps → Develop apps
   → Create app → Configure Storefront API
   → Enable permissions → Install
   → Copy token
   ```

4. **Configure Website**
   ```bash
   # Create .env.local file
   VITE_SHOPIFY_DOMAIN=yourstore.myshopify.com
   VITE_SHOPIFY_STOREFRONT_ACCESS_TOKEN=your_token
   ```

5. **Test**
   ```bash
   npm start
   # Visit http://localhost:5173/#shop
   ```

**That's it!** Your shop should be working. 🎉

For details, see [ECOMMERCE_QUICKSTART.md](ECOMMERCE_QUICKSTART.md)
